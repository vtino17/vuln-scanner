# Changelog

## 1.0.0 (2026-07-22)

### Added

- Initial release of vuln-scanner
- TCP port scanning with service detection
- CIDR and hostname target support
- Banner grabbing for service version detection
- TLS certificate inspection
- CVE correlation via NVD API
- JSON report output
- Plugin system for extensible checks
- Slack webhook notifications
- Configurable scan rate and timeout
- Multi-threaded concurrent scanning
- Unit test suite with pytest
